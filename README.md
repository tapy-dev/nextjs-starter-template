# https://tapy.to nextjs starter template

The template ships with **Tailwind**, **`src/components/ui` (shadcn-style primitives)**, and common libraries so AI-generated sites can import components without running extra installs first.

## Form integration

Forms use the **`submitTapyForm`** helper — see **[docs/FORM.md](./docs/FORM.md)** for attributes, pipeline `<meta>` flow, errors, and local testing.
